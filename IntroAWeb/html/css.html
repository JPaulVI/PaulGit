CSS

CSS significa Cascading Style Sheets, indica como van a estar colocados los elementos de HTML

Sintaxis de CSS

CSS consiste en dos boques, uno llamado "selector" y otro llamado "declaration". Ej: h1 {color:blue; font-size:12px;} Donde "color" y "font-size" son propiedades y "blue" y "declaration" son valores. Las declaraciones contienen punto-comas (";"). Cada declaración contiene una propiedad y un valor separados por una columna. Los bloques de declaración deben de estar entre llaves ("{" y "}"). Los selectores son elementos de CSS para seleccionar elementos de HTML tipo name, id, class, attributes, etc.

Selectors

Element selector

Con el element selector puedes seleccionear los elementos de HTML a partir de su nombre ej: p { text-align: center; color: red; }

Id selector

Con el id selector puedes seleccionar los elemento de HTLM a partir de su id con tan solo poner "#" antes del id ej: #para1 { text-align: center; color: red; }

Class selector

Con el class selector puedes seleccionar los elemento de HTLM a partir de su clase con tan solo poner "." antes de la clase ej: .center { text-align: center; color: red; } También puedes especifocar un colo elemnto de una clase ej: p.center { text-align: center; color: red; } O a elementos de más de una clase ej: <p class="center large">This paragraph refers to two classes.</p>

Agrupar selectors

Puedes agrupar selectors para minimizar codigo separandolos con una coma (",") ej: h1, h2, p { text-align: center; color: red; }

Comentarios en CSS

Puedes añadir comentarios con CSS usan "/*" antes del comentario y "*/" después del comentario.

Como usar CSS

External Style Sheet

Puedes crear un archivo de CSS usando ".css" y referenciarlo en el documento HTML ej: <head> <link rel="stylesheet" type="text/css" href="mystyle.css"> </head>

Internal Style Sheet

Se suele estilisar internamente los diseños de un documento si este tiene un estilo único. para eso se usa el tag "style" debajo del tag "head" ej: <head> <style> body { background-color: linen; } h1 { color: maroon; margin-left: 40px; } </style> </head>

Inline Style Sheet

Se puede añadir css a un solo elemento de HTML ej: <h1 style="color:blue;margin-left:30px;">This is a heading</h1>

Prioridad en los Styles Sheets

  • Inline style (inside an HTML element)
  • External and internal style sheets (in the head section)
  • Browser default

Background

Background color: body { background-color: lightblue; } Background image: body { background-image: url("paper.gif"); } Ciertos backrounds se tienen que repetir verticamente u horizontalmente para que se vean bien: body { background-image: url("gradient_bg.png"); background-repeat: repeat-x; } Si no quieres que se repita una imgen: body { background-image: url("img_tree.png"); background-repeat: no-repeat; } Para posicionar un imagen: body { background-image: url("img_tree.png"); background-repeat: no-repeat; background-position: right top; } Para que la imagen se quede fija: body { background-image: url("img_tree.png"); background-repeat: no-repeat; background-position: right top; background-attachment: fixed; } Shorthand property (Para minimizar el código): body { background: #ffffff url("img_tree.png") no-repeat right top; }

Bordes

Distintos bordes:

  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value
  • inset - Defines a 3D inset border. The effect depends on the border-color value
  • outset - Defines a 3D outset border. The effect depends on the border-color value
  • none - Defines no border
  • hidden - Defines a hidden border
Ejeplos de comom añadir bordes: p.dotted {border-style: dotted;} p.dashed {border-style: dashed;} p.solid {border-style: solid;} p.double {border-style: double;} p.groove {border-style: groove;} p.ridge {border-style: ridge;} p.inset {border-style: inset;} p.outset {border-style: outset;} p.none {border-style: none;} p.hidden {border-style: hidden;} p.mix {border-style: dotted dashed solid double;}

Tamaño del borde

Se puede seleccionar el tamaño del borde unidades (px, cm, pt, em) o por valores predeterminados: thin, medium, or thick. Ej: p.one { border-style: solid; border-width: 5px; } p.two { border-style: solid; border-width: medium; } p.three { border-style: solid; border-width: 2px 10px 4px 20px; } Se le puede añadir color al borde con border-color. Ej: p.one { border-style: solid; border-color: red; } p.two { border-style: solid; border-color: green; } p.three { border-style: solid; border-color: red green blue yellow; } (The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border)

Bordes individuales

Cada borde puede tener estilo y color unico ej: p { border-top-style: dotted; border-right-style: solid; border-bottom-style: dotted; border-left-style: solid; } p { border-style: dotted solid; }

Shorthand Property

Se puede minizar el codigo con la shorthand property con los siguientes atributos:

  • border-width
  • border-style (required)
  • border-color
Ej: p { border: 5px solid red; } Puedes estilizar un solo un borde ej: p { border-left: 6px solid red; background-color: lightgrey; }

border-radius

Se usa border-radius para crear un borde redondo ej: p { border: 2px solid red; border-radius: 5px; }

Margenes

CSS tiene las siguientes propiedades para cada elemento:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
Los cuales pueden tener los siguientes valores:
  • auto: el navegador lo calcula
  • ength: especificado en px, pt, cm, etc
  • %: especificando el margen en porcentage del elemento contenido
  • inherit: le dice al elemento que tiene que heredar el margen de su padre
Ej: p { margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px; }

Shorthand Property

Se pueden especificar todas las propiedades de un margen en una sola propiedad ej: p { margin: 100px 150px 100px 80px; }

Padding

CSS tiene las siguientes propiedades para cada elemento:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
Los cuales pueden tener los siguientes valores:
  • lenght
  • %
  • inherit
Ej: p { padding-top: 50px; padding-right: 30px; padding-bottom: 50px; padding-left: 80px; }

Shorthand Property

Ej:p { padding: 50px 30px 50px 80px; } div.ex1 { padding: 25px 50px 75px 100px; } div.ex2 { padding: 25px 50px 75px; } div.ex3 { padding: 25px 50px; } div.ex4 { padding: 25px; }

Alto y ancho

Con height y width le das tamaña al elemento (ya viene una propoción por omisión) div { height: 200px; width: 50%; background-color: powderblue; } con max-height y max-width le dices al elemento cual será su máximo tamaño.

Modelo Box

En HTML un elemento se puede considerar como cajas: Consiste de las siguientes partes:

  • Content: Es el contenido del elemento (donde las imágenes y el texot aparecen
  • Padding: Crea un espacio transparente al rededor del contenido
  • Border: Un borde que va al rededor del padding y el content
  • Margin: Crea un area afuera del borde
Ej: div { width: 300px; border: 25px solid green; padding: 25px; margin: 25px; } Para calcular la anchura de un elemento se usa la siguiente formula: Ancho total de un elemento = width + left padding + right padding + left border + right border + left margin + right margin Y para la altua de un elemento: Altura total de un elemento = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

Outline

Outline es una linea afuera del borde de un elemento (se usa para resaltar un elemento de los demás). outline-stlye es la propiedad que especifica el estilo que va a llevar el outline. Valores posibles:

  • dotted - Defines a dotted outline
  • dashed - Defines a dashed outline
  • solid - Defines a solid outline
  • double - Defines a double outline
  • groove - Defines a 3D grooved outline. The effect depends on the outline-color value
  • ridge - Defines a 3D ridged outline. The effect depends on the outline-color value
  • inset - Defines a 3D inset outline. The effect depends on the outline-color value
  • outset - Defines a 3D outset outline. The effect depends on the outline-color value
  • none - Defines no outline
  • hidden - Defines a hidden outline
Ej: p { border: 1px solid black; outline-color: red; } p.dotted {outline-style: dotted;} p.dashed {outline-style: dashed;} p.solid {outline-style: solid;} p.double {outline-style: double;} p.groove {outline-style: groove;} p.ridge {outline-style: ridge;} p.inset {outline-style: inset;} p.outset {outline-style: outset;} Para las siguientes propiedades se necisita declarar primero la propiedad outline-style. outline-color ej: p { border: 1px solid black; outline-style: double; outline-color: red; } outline-width ej: p {border: 1px solid black;} p.one { outline-style: double; outline-color: red; outline-width: thick; } p.two { outline-style: double; outline-color: green; outline-width: 3px; } Outline tiene Shorthand property Ej: p { border: 1px solid black; outline: 5px dotted red; }

Texto

Color

Con la propiedad color le das color al texto ej: body { color: blue; } h1 { color: green; }

Text Alignment

Con la propiedad text-align alineas horizonatalmente el texto. Valores posibles: right, left, centered, justified ej: h1 { text-align: center; } h2 { text-align: left; } h3 { text-align: right; }

Text decoration

Con la propiedad text-decoration se usa para quitar o pones docoraciones en el texto. Valores posibles: none(le quitas la decoración al texto), overline, line-through, underline. Ej: h1 { text-decoration: overline; } h2 { text-decoration: line-through; } h3 { text-decoration: underline; }

Text transformation

Con la propiedad text-transform le dices al texto como se va a mostrar. Valores posible: uppercase, lowercase, capitalize. Ej: p.uppercase { text-transform: uppercase; } p.lowercase { text-transform: lowercase; } p.capitalize { text-transform: capitalize; }

Text indentation

Con la propiedad text-indent marcas la sangría del texto. Ej: p { text-indent: 50px; }

Letter spacing

Con la propiedad letter-spacing defines el espacio entre los carácteres. Ej: h1 { letter-spacing: 3px; } h2 { letter-spacing: -3px; }

Line Height

Con la propiedad line-height le defines el espacio entre las lineas. Ej: p.small { line-height: 0.8; } p.big { line-height: 1.8; }

Text direction

Con la propiedad direction defines la direccion del texto. Ej: div { direction: rtl; } (rtl significa rigth to left)

Word Spacing

Con el word-spacing defines el epacio entre las palabras. Ej: h1 { word-spacing: 10px; } h2 { word-spacing: -5px; }

Fuentes

Font Family

p { font-family: "Times New Roman", Times, serif; }

Font Style

Con la propiedad font-style decides si va en italicas o no. Valores posibles: normal, italic, oblique Ej: p.normal { font-style: normal; } p.italic { font-style: italic; } p.oblique { font-style: oblique; }

Font size

Con el parametro font-size defines el tamaño del texto. Ej: h1 { font-size: 40px; } h2 { font-size: 30px; } p { font-size: 14px; } puedes usar pixeles(px) o em (Nota: 16px = 1em).

Font Weight

Tribial ej: p.normal { font-weight: normal; } p.thick { font-weight: bold; }

Font Variant

p.normal { font-variant: normal; } p.small { font-variant: small-caps; }

Media Queries

Sintaxis

@media not|only mediatype and (expressions) { CSS-Code; }

Viewport

Con el viewport puedes escalar tu documento. Solo necesitas añaidir lo siguirte a tu firma: <meta name="viewport" content="width=device-width, initial-scale=1.0"> Algunas reglas a tomar en cuenta:

  • No usar elemento fijos con un ancho largo.
  • No dejar que los elemtos queden bien en un solo viewport.
  • Usa medai queries para adaptar los estilos en las pequeñas y grandes pantallas.

Gridview

Muchas páginas usan el gridview para ser diseñadas, lo que significa que esta dividida en muchas columnas. Nota: usualmente las página cuenta con 12 columnas y un tamaño de 100% del tamaño del navegador.

Cómo construir un gridview

Asegurate que tus elementos de CSS tienen la propiedad: "box-sizing" con valor "border-box" Ej: * { box-sizing: border-box; } luego se crean 12 columnas :class="col-". Ej: .col-1 {width: 8.33%;} .col-2 {width: 16.66%;} .col-3 {width: 25%;} .col-4 {width: 33.33%;} .col-5 {width: 41.66%;} .col-6 {width: 50%;} .col-7 {width: 58.33%;} .col-8 {width: 66.66%;} .col-9 {width: 75%;} .col-10 {width: 83.33%;} .col-11 {width: 91.66%;} .col-12 {width: 100%;} Todas estas columnas deben de tener un el floar a la izq y un padding de 15px. [class*="col-"] { float: left; padding: 15px; border: 1px solid red; } <div class="row"> <div class="col-3">...</div> <div class="col-9">...</div> </div> Todos los renglones tienen un flow que hay que limpiar con las siguientes instrucciones: .row::after { content: ""; clear: both; display: table; }

Media queries (continuación)

Con los media queries puedes modificar ciertos elementos del documento cuando el viewport cambia de tamaño. Es muy recomendado diseñar la versión móvil primero. Ej: /* For mobile phones: */ [class*="col-"] { width: 100%; } @media only screen and (min-width: 600px) { /* For tablets: */ .col-m-1 {width: 8.33%;} .col-m-2 {width: 16.66%;} .col-m-3 {width: 25%;} .col-m-4 {width: 33.33%;} .col-m-5 {width: 41.66%;} .col-m-6 {width: 50%;} .col-m-7 {width: 58.33%;} .col-m-8 {width: 66.66%;} .col-m-9 {width: 75%;} .col-m-10 {width: 83.33%;} .col-m-11 {width: 91.66%;} .col-m-12 {width: 100%;} } @media only screen and (min-width: 768px) { /* For desktop: */ .col-1 {width: 8.33%;} .col-2 {width: 16.66%;} .col-3 {width: 25%;} .col-4 {width: 33.33%;} .col-5 {width: 41.66%;} .col-6 {width: 50%;} .col-7 {width: 58.33%;} .col-8 {width: 66.66%;} .col-9 {width: 75%;} .col-10 {width: 83.33%;} .col-11 {width: 91.66%;} .col-12 {width: 100%;} }
