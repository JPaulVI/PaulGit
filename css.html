CSS

CSS significa Cascading Style Sheets, indica como van a estar colocados los elementos de HTML

Sintaxis de CSS

	CSS consiste en dos boques, uno llamado "selector" y otro llamado "declaration".
	Ej: h1 {color:blue; font-size:12px;}
	Donde "color" y "font-size" son propiedades y "blue" y "declaration" son valores.
	Las declaraciones contienen punto-comas (";").
	Cada declaración contiene una propiedad y un valor separados por una columna.
	Los bloques de declaración deben de estar entre llaves ("{" y "}").
	Los selectores son elementos de CSS para seleccionar elementos de HTML tipo name, id, class, attributes, etc.

Selectors

Element selector

	Con el element selector puedes seleccionear los elementos de HTML a partir de su nombre ej:
	p {
	    text-align: center;
	    color: red;
	}

Id selector

	Con el id selector puedes seleccionar los elemento de HTLM a partir de su id con tan solo poner "#" antes del id ej:
	#para1 {
    text-align: center;
    color: red;
	}

Class selector

	Con el class selector puedes seleccionar los elemento de HTLM a partir de su clase con tan solo poner "." antes de la clase ej:
	.center {
    text-align: center;
    color: red;
	}
	También puedes especifocar un colo elemnto de una clase ej:
	p.center {
    text-align: center;
    color: red;
	}
	O a elementos de más de una clase ej:
	<p class="center large">This paragraph refers to two classes.</p>

Agrupar selectors

	Puedes agrupar selectors para minimizar codigo separandolos con una coma (",") ej:
	h1, h2, p {
    text-align: center;
    color: red;
	}

Comentarios en CSS

Puedes añadir comentarios con CSS usan "/*" antes del comentario y "*/" después del comentario.

Como usar CSS

External Style Sheet

	Puedes crear un archivo de CSS usando ".css" y referenciarlo en el documento HTML ej:
	<head>
	<link rel="stylesheet" type="text/css" href="mystyle.css">
	</head>

Internal Style Sheet

	Se suele estilisar internamente los diseños de un documento si este tiene un estilo único.
	para eso se usa el tag "style" debajo del tag "head" ej:
	<head>
	<style>
	body {
	    background-color: linen;
	}

	h1 {
	    color: maroon;
	    margin-left: 40px;
	} 
	</style>
	</head>

Inline Style Sheet

	Se puede añadir css a un solo elemento de HTML ej:
	<h1 style="color:blue;margin-left:30px;">This is a heading</h1>

Prioridad en los Styles Sheets

  • Inline style (inside an HTML element)
  • External and internal style sheets (in the head section)
  • Browser default

Background

	Background color:

	body {
    background-color: lightblue;
	}

	Background image:
	body {
    background-image: url("paper.gif");
	}

	Ciertos backrounds se tienen que repetir verticamente u horizontalmente para que se vean bien:
	body {
    background-image: url("gradient_bg.png");
    background-repeat: repeat-x;
	}

	Si no quieres que se repita una imgen:
	body {
    background-image: url("img_tree.png");
    background-repeat: no-repeat;
	}

	Para posicionar un imagen:
	body {
    background-image: url("img_tree.png");
    background-repeat: no-repeat;
    background-position: right top;
	}

	Para que la imagen se quede fija:
	body {
    background-image: url("img_tree.png");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: fixed;
	}

	Shorthand property (Para minimizar el código):
	body {
    background: #ffffff url("img_tree.png") no-repeat right top;
	}

Tamaño del borde

	Se puede seleccionar el tamaño del borde unidades (px, cm, pt, em) o por valores predeterminados: thin, medium, or thick.
	Ej:
	p.one {
    border-style: solid;
    border-width: 5px;
	}

	p.two {
	    border-style: solid;
	    border-width: medium;
	}

	p.three {
	    border-style: solid;
	    border-width: 2px 10px 4px 20px;
	}
	Se le puede añadir color al borde con border-color.
	Ej:
	p.one {
    border-style: solid;
    border-color: red;
	}

	p.two {
	    border-style: solid;
	    border-color: green;
	}

	p.three {
	    border-style: solid;
	    border-color: red green blue yellow;
	}
	(The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border)

Bordes individuales

	Cada borde puede tener estilo y color unico ej:
	p {
    border-top-style: dotted;
    border-right-style: solid;
    border-bottom-style: dotted;
    border-left-style: solid;
	}
	p {
    border-style: dotted solid;
	}
