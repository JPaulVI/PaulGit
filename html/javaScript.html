JavaScript

Introcción

JavaScript es el lenguaje de programación que te permite alterar el comportamiento de tu página web.

Dónde escribir JavaScript

para poder usar JavaScript se debe de insertar en el tag "script" de html y se puede insertar tanto en head como en body.

También se pueden escribir las funciones de JavaScript en archivos externos.
Ej: function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
Y se referencian así:
<!DOCTYPE html>
<html>
>body>

<script src="myScript.js"></script>

</body>
</html>
Los archivos JavaScript tienen como extensión ".js".

Outputs

Hay cuatro tipos de output en JS:

  • innerHTML
  • .write()
  • window.alert()
  • console.log()

Sintaxis

En JS las declaraciones se hacen con ";" al final.
Tiene "literals" (fijas) y variables.
También tiene operadores expresiones y keywords.
Las variables se declaran de la siguiente forma:
var nobreDeVariable = valor;

DataTypes

Hay tres tipos de datos rincipales.

  • Number
  • String
  • Object

Funciones

Las funciones de daclaran de la suiguiente forma:

//código de la funcción
}

Objetos

Ejemplo de un objeto:
var car = {type:"Fiat", model:"500", color:"white"};

Eventos

Estos son los eventos más comunes en JS:

Event Description
onchange An HTML element has been changed
onclick The user clicks an HTML element
onmouseover The user moves the mouse over an HTML element
onmouseout The user moves the mouse away from an HTML element
onkeydown The user pushes a keyboard key
onload The browser has finished loading the page
