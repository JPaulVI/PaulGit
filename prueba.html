Introducción

Todo en HTML jala con "tags" de este tipo: "<"tagname">"content goes here..."<"/tagname">" siempre tiene un opening tag y un closing tag

!Doctype

!Doctype le dice a tu navegador como mostrar las páginas correctamente. Así se declara: "<"!DOCTYPE html">"

Editores

Existen varios editores, pero cualquier editor de texto jala. Solo no olvides de ponerlo en un archivo .html

Documentos HTML

Todos los documentos HTML tinen que tener el "<"!DOCTYPE html">", empezar con "<"html">" y terminar con "<"/html">" y, por supuesto, su parte visible: "<"body">" ... "<"/body">"

Headings

Los Headings son tags de los titulos y van del "h1" al "h6" dependiendo de su importancia

Parrafos

Para declarar un parrafo se usa el tag "p"

Ligas

Para declarar una liga de usa el tag "a" seguido del atributo "href"

Puedes usar un link en imgenes, texto, etc... ej: <a href="url">link text</a>

Imágenes

Para declarar una imagen se usa el tag "img" y puede tener los siguientes atributos: fuente "scr", texto alterno "alt, largo "width" y ancho "height"

Sintaxis para declarar imágenes

	Para declarar una imagen HTML no se necesita un closing tag.
	Pudes insertar imagen animadas y no animadas en tu documento.
	Esta es la sintaxis: <img src="url" alt="some_text" style="width:width;height:height;">

Mapa en un imagen

Ejemplo de como poner una imagen en un mapa: 
	<img src="planets.gif" alt="Planets" usemap="#planetmap" style="width:145px;height:126px;">
		<map name="planetmap">
		<area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm">
		<area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm">
		<area shape="circle" coords="124,58,8" alt="Venus" href="venus.htm">
	</map>

Elementos de HTML

HTML tiene varios elementos como "html" el cual define el tipo de documento y se tiene que cerrar, "body" define lo que tendra el documento y se tiene que cerrar, y hay elementos de tipo "content" que los que contendra el documento, estos ultimo de como "p", "h2" y se tienen que cerrar. Es muy importante cerrar estos tags. Hay ciertos tipos de tags llamados "empty" los cuales no tienen contenido y no se tienen que cerrar el como el break("br"). A HTML le vale que uses mayúsculas, pero es mejor no usarlas.

Atributos

HTML puede tener atributos los cuales añaden información adicional sobre un elemento. Los at ributos siempre se especifican el los tags de inicio

Atributos

Los elementos de HTML pueden atributos, los cuales añaden informacion adicional al elemento. Los atributos simepre se especifican en el start tag. Los atributos se declacan de la suiguiente manera: name="value".

Ejemplos de atributos:

  • lang: es el idioma ej: <html lang="en-US">
  • title: sale como tooltip cuando pasas el mouse como parrafo ej: <p title="I'm a tooltip">.
  • href: es la referencia hacia un link ej: <a href="https://www.w3schools.com">
  • src: es para la fuente de un imagen
  • width: es para el ancho de una imagen
  • height: es para le largo imagen
  • Ej: <img src="w3schools.jpg" width="104" height="142" >
  • para el ancho y el largo la medida el en pixeles
  • alt: es la descripción de la imagen, además se muestra en caso de que la imagen mo se pueda mostrar este texto aparecera ej: <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142"&dt;
  • alt Specifies an alternative text for an image, when the image cannot be displayed disabled Specifies that an input element should be disabled href Specifies the URL (web address) for a link id Specifies a unique id for an element src Specifies the URL (web address) for an image style Specifies an inline CSS style for an element title Specifies extra information about an element (displayed as a tool tip)

Atributo Target

Con el atributo target, le especificas al documento como queres que se abran los links

  • _blank: abre la liga en una nueva ventana
  • _self: abre la liga en la misma venatana (esta es la ventana que viene por default)
  • _parent: abre la liga en una parent frame
  • _top: abre la lia en una ventana completa
  • framename: abre la liga en la frame nombrada
  • Ej:<a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>
  • Ejemplo de como ponerle una liga a una imagen: <a href="default.asp"> <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;border:0;"> </a>

Bookmark

Los bookmark se usan para saltar a una parte especifica de un documento ej: se añade la bookmark <h2 id="C4">Chapter 4</h2> y luego se le añade la liga <a href="#C4">Jump to Chapter 4</a>

Ejemplo de como usar una liga con un path <a href="https://www.w3schools.com/html/default.asp">HTML tutorial</a>

Tablas

Definir una tabla

	Para definir una tabal se usa el tag "table" y se tiene que cerrar.
	Cada renglon se define ocon el tag "tr" y se tiene que cerrar.
	Cada celda se define con el tag "td" y se tiene que cerrar, puede con tener imagenes, texto, otra tabla, etc...
	Existen los siguientes tags:
	colspan: se usa para declara una celda que ocupa mas de una columna
	rowspan: se usa para declarar una celda que ocupa mas de una renglón
	Se usa el tag "caption" para añadir un titulo a la tabla

Listas

Lista desordenada

La lista desordena se crea usando el tag "ul" y cada elemento de la lista se crea usando el tag "li"

Lista ordenada

La lista ordenada se crea con el tal "ol" y cada elemento de la lita se crea usnado el tag "li" (la lista ordenada le añade numeros a los elementos)
